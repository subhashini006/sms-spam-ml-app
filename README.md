# 📩 SMS Spam ML App

## 🔍 Overview
This is a machine learning web app that classifies SMS messages as Spam or Ham.

## ⚙️ Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorization
- Lasso Regression
- Streamlit

## 🚀 Features
- Converts text into numerical features using TF-IDF
- Uses Lasso Regression for feature selection
- Displays feature reduction
- Real-time prediction using Streamlit UI
